# 帮我评评理 (Pil) - UI/UX 重构设计 (v3 - 广场流式交互)

基于用户反馈的修正版：**无弹窗交互，广场信息流模式，强化 AI 分身参与感。**

## 核心交互逻辑
1.  **广场主页 (The Square)**: 展示所有用户的提问流。
2.  **流式参与**: 提问卡片展示实时的 AI 辩论进度（红蓝条 + 最新回复）。
3.  **无弹窗体验**: 点击卡片直接进入“详情模式”或在当前流中展开，避免打断感的 Modal。

---

## 页面布局 (Markdown Mockup)

### 1. 顶部：发布器 (The Publisher)
保持极简，鼓励用户发起新的讨论。
```markdown
+------------------------------------------------------+
|  [ User Avatar ]  "有什么想让 AI 评评理？"             |
|  bg-white rounded-3xl p-4 shadow-sm border-stone-100 |
|  flex items-center gap-4 cursor-pointer              |
|  (点击展开完整输入框)                                  |
+------------------------------------------------------+
```

### 2. 内容流 (The Feed)
按时间倒序排列的卡片流。

**单张卡片 (Feed Card)**
```markdown
+------------------------------------------------------+
|  bg-white rounded-3xl p-6 mb-4 border border-stone-50|
|                                                      |
|  [ Header ]                                          |
|  [Avatar] User Name · 2h ago                         |
|  text-xs text-stone-400                              |
|                                                      |
|  [ Content ]                                         |
|  "相亲男让我AA这杯30块的咖啡..."                         |
|  text-base text-stone-900 font-medium mb-4           |
|                                                      |
|  [ Arena Visual ]                                    |
|  [======(红 70%)======|===(蓝 30%)===]               |
|  h-2 rounded-full bg-stone-100 mb-4                  |
|                                                      |
|  [ AI Comments Preview ]                             |
|  [Avatar] 毒舌分身: "转给他，别惯着..."                  |
|  [Avatar] 理性分身: "从经济学角度..."                    |
|  text-xs text-stone-600 bg-stone-50 rounded-lg p-2   |
|                                                      |
|  [ Actions ]                                         |
|  [MessageCircle] 12条评论    [Share] 分享判决          |
|  text-stone-400 hover:text-stone-900                 |
|                                                      |
+------------------------------------------------------+
```

### 3. 图标系统 (Lucide React)
- **场域**: `Flame` (毒舌), `HeartHandshake` (安慰), `Brain` (理性)
- **操作**: `MessageSquare` (评论), `Share2` (分享), `Send` (发布)
- **状态**: `Activity` (进行中), `CheckCircle2` (已结案)

## 实施计划 (Implementation)

1.  **组件重构**:
    -   `PilPage`: 改为 Feed 流布局。
    -   `QuestionInput`: 改为顶部折叠式/展开式卡片。
    -   `FeedCard` (新组件): 展示单个问题的状态和 AI 回复预览。
    -   `ArenaVisual` (重构): 适配极简细条风格。
2.  **样式升级**:
    -   引入 `lucide-react`。
    -   应用 `DESIGN.md` 配色 (Stone/Rose/Slate)。
3.  **逻辑对接 (Mock for UI)**:
    -   由于后端异步逻辑复杂，UI 层先使用 Mock 数据展示“多 AI 回复”的效果，确保视觉还原。

*注：后端逻辑（AI 批量回复/Prompt 优化）将在 UI 框架定型后，在 API 层单独处理。*
